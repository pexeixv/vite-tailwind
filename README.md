# ViteJS + TailwindCSS

Starter Template by [Gavin Pereira](http://gavn.in/)

## Cloning

```console
git clone https://github.com/pexeixv/vite-tailwind
```

## Install dependencies

```console
pnpm i
```

## Run Development Server

```console
pnpm dev
```

## Build for Production

```console
pnpm build
```
